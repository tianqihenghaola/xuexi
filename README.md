# xuexi

随便做做的vue学习项目

## 目录结构说明

vue-integrate

- public

- src
  - assets - 共用的静态资源
  - components - 共用的组件
  - layout - 主页的布局容器
  - plugins - 使用的插件
  - router - 路由配置
  - styles - 全局的样式配置
  - views - 主业务的页面路径
- .browserslistrc
- .eslintrc.js - eslint 配置文件
- .gitignore - git 忽略文件
- babel.config.js - babel 编译配置文件
- LICENSE
- package.json
- package-lock.json
